The Problem

Rehabilitasi rumahan di Indonesia itu masih rusak.

Jutaan pasien pasca-rehabilitasi dipulangkan dengan selembar kertas instruksi. Tidak ada panduan visual, tidak ada umpan balik, tidak ada cara mengukur apakah mereka melakukan gerakan dengan benar.

Riset Kesehatan Dasar (Riskesdas) 2023
"Pasien dipulangkan dengan kertas instruksi. Tanpa video. Tanpa umpan balik. Tanpa cara tahu apakah gerakannya benar. Akibatnya, cederanya kambuh, progress stagnan, dan kualitas hidup tidak membaik."
Dokter spesialis ortopedi biasanya hanya punya 10–15 menit per pasien di klinik. Tidak ada waktu untuk mengajarkan setiap latihan secara mendalam. Platform digital yang memandu pasien di rumah bukan sekadar bagus — ini kebutuhan medis nyata.

Technology

Dibangun di atas teknologi terpercaya

🐍
Flask + Python
Backend lightweight dengan pattern MVC yang bersih. Blueprint-based routing, SQLAlchemy ORM, session-based auth.
🗄️
SQLAlchemy + SQLite
ORM relasional yang teruji. Dev menggunakan SQLite, produksi siap migrasi ke PostgreSQL dengan zero code changes.
🤸
MediaPipe (In Dev)
Google's MediaPipe Pose melacak 33 landmark tubuh per frame via WebAssembly — semua berjalan di sisi klien, privasi terjaga.
🛡️
Security First
XSS prevention, password hashing PBKDF2-SHA256, open redirect protection, session hardening, input validation ketat.
🧪
pytest Test Suite
Coverage lengkap: auth flows, validator, route tests, session complete, accuracy clamping — siap CI/CD pipeline.
☁️
Cloud Ready
Siap deploy ke AWS/GCP/Heroku dengan Docker containerization. Konfigurasi per-environment yang bersih.
